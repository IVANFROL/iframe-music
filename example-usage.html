Radio Nostalgie - Iframe Плеер

Код для встраивания на ваш сайт:

<iframe src="iframe.html" width="350" height="400" frameborder="0"></iframe>

Управление плеером через JavaScript:

// Получить iframe
const iframe = document.querySelector('iframe');

// Воспроизвести
iframe.contentWindow.postMessage({action: 'play'}, '*');

// Остановить
iframe.contentWindow.postMessage({action: 'stop'}, '*');

// Установить громкость (0-100)
iframe.contentWindow.postMessage({action: 'setVolume', volume: 70}, '*');

Особенности:

  • ✅ Адаптивный дизайн - подстраивается под размер iframe
  • ✅ Современный UI с градиентами и анимациями
  • ✅ Управление громкостью
  • ✅ Статус подключения в реальном времени
  • ✅ Обложка трека с hover эффектами
  • ✅ Поддержка мобильных устройств
  • ✅ Обработка ошибок подключения
