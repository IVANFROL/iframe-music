🎵 Radio Nostalgie - Iframe Код для вставки

Предварительный просмотр компактного плеера:

📋 Основной код iframe (компактный):

<iframe src="https://iframe-music.vercel.app/iframe-compact.html" width="400" height="80" frameborder="0"></iframe>

🎛️ Управление плеером через JavaScript:

// Получить iframe элемент
const iframe = document.querySelector('iframe');

// Воспроизвести
iframe.contentWindow.postMessage({action: 'play'}, '*');

// Остановить
iframe.contentWindow.postMessage({action: 'pause'}, '*');

// Установить громкость (0-100)
iframe.contentWindow.postMessage({action: 'setVolume', volume: 70}, '*');
